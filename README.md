# A Tour of C++ 第三版

在线阅读：https://heizer-tan.github.io/a-tour-of-cpp-3rd-cn/

## 简介

本书为 **[A Tour of C++ (Third edition)](https://www.stroustrup.com/tour3.html)** 的中文翻译，概述了由 C++20 所定义的现代 C++ 语言特性和标准库组件。
