# 第一章 基础知识

## 1.1 引言

C++ 是一种通用的、静态类型的、编译型编程语言。它支持多种编程范式，包括过程式、面向对象、泛型和函数式编程。

## 1.2 程序

C++ 是一种编译型语言。要运行程序，其源代码必须经过编译器处理，生成目标文件，然后由链接器组合成可执行程序。一个 C++ 程序通常由多个源代码文件组成（通常简称为源文件）。

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!\n";
    return 0;
}
```

## 1.3 函数

函数是执行特定任务的代码块。C++ 程序从 `main()` 函数开始执行。

## 1.4 类型、变量和算术运算

C++ 是一种静态类型语言，每个实体（如对象、值、名称和表达式）的类型在使用点必须为编译器所知。

## 1.5 作用域和生命周期

作用域是程序的一部分，在其中名称可以被访问。生命周期是对象存在的时间段。

## 1.6 常量

常量是在程序执行期间值不变的对象。

## 1.7 指针、数组和引用

指针是存储内存地址的变量。数组是相同类型元素的连续序列。引用是对象的别名。

## 1.8 测试

条件语句允许程序根据条件执行不同的代码路径。

## 1.9 映射到硬件

C++ 提供了直接访问硬件的能力，同时保持高级语言的抽象。

## 1.10 建议

- 优先使用 `constexpr` 而非宏常量；
- 避免裸指针，优先使用智能指针；
- 使用范围 `for` 循环遍历容器；
- 优先使用 `auto` 推导类型。
