# 第二章 用户定义类型

## 2.1 引言 {#21}

用户定义类型允许程序员创建适合特定问题领域的数据结构。

## 2.2 结构体 {#22}

结构体是将多个不同类型的数据项组合在一起的聚合类型。

```cpp
struct Point {
    double x;
    double y;
};
```

## 2.3 类 {#23}

类是封装数据和操作数据的函数的组合。

## 2.4 枚举 {#24}

枚举是用户定义的整数类型，其值由一组命名常量组成。

## 2.5 联合体 {#25}

联合体允许在同一内存位置存储不同类型的数据。

## 2.6 建议 {#26}

- 使用结构体表示简单的数据聚合；
- 使用类封装复杂的状态和行为；
- 优先使用强类型枚举 `enum class`。
