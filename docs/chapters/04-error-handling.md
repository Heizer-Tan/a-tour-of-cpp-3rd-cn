# 第四章 错误处理

## 4.1 引言

错误处理是编写健壮软件的关键部分。

## 4.2 异常

异常提供了一种处理运行时错误的机制。

```cpp
try {
    // 可能抛出异常的代码
} catch (const std::exception& e) {
    // 处理异常
}
```

## 4.3 不变式

不变式是对象在其生命周期内始终保持为真的条件。

## 4.4 错误处理替代方案

除了异常，还可以使用错误码、断言等方式处理错误。

## 4.5 断言

断言用于检查程序中的假设。

## 4.6 建议

- 对不可恢复的错误使用断言；
- 对可恢复的错误使用异常或错误码；
- 提供清晰的错误信息。
