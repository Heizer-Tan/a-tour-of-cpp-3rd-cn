# 第十一章 输入输出

## 11.1 引言

输入输出是程序与外部世界交互的方式。

## 11.2 输出

`std::cout` 用于标准输出。

## 11.3 输入

`std::cin` 用于标准输入。

## 11.4 I/O 状态

I/O 操作可能失败，需要检查状态。

## 11.5 用户定义类型的 I/O

可以为用户定义类型重载 `<<` 和 `>>` 运算符。

## 11.6 输出格式

可以控制输出的格式。

## 11.7 流

流是 I/O 的抽象。

## 11.8 C 风格 I/O

C++ 也支持 C 风格的 I/O。

## 11.9 文件系统

`std::filesystem` 提供了文件系统操作。

## 11.10 建议

- 优先使用 C++ I/O 而非 C 风格 I/O；
- 使用 `std::filesystem` 处理文件系统。
