# 1.7 指针、数组和引用

## 1.7.1 数组

最基本的数据集合是数组（array），即一组连续分配的同类型元素。例如：

```cpp
char v[6];      // 6 个字符的数组
```

数组的大小必须是常量表达式。数组的下标从 0 开始，因此 `v` 包含 `v[0]` 到 `v[5]`。访问数组时，C++ 不进行范围检查，越界访问会导致未定义行为。

可以这样初始化数组：

```cpp
char v[] = { 'a', 'b', 'c', 'd', 'e', '\0' };
```

字符串字面量本质上是字符数组：

```cpp
char v[] = "abcde";  // 等价于上面的初始化，末尾自动添加 '\0'
```

## 1.7.2 指针

指针（pointer）是一个变量，存储着某个对象的地址：

```cpp
char* p = &v[3];    // p 指向 v 的第四个元素
char x = *p;        // *p 是 p 指向的对象，即 v[3]
```

`&` 是取地址运算符，`*` 是解引用运算符。`*p` 表示 `p` 所指向的对象。

指针可以指向数组元素，也可以指向独立的对象。`nullptr` 表示空指针，即不指向任何对象：

```cpp
int* p = nullptr;   // p 不指向任何对象
```

在较早的代码中，常用 `0` 或 `NULL` 代替 `nullptr`。使用 `nullptr` 可以避免整数和指针之间的混淆。

指针可以进行算术运算：

```cpp
char* p = &v[0];
++p;                // p 现在指向 v[1]
p += 2;             // p 现在指向 v[3]
```

## 1.7.3 引用

引用（reference）是对象的别名。引用一旦初始化，就不能再引用其他对象。例如：

```cpp
int x = 42;
int& r = x;         // r 是 x 的引用
r = 7;              // 修改 r 就是修改 x，现在 x == 7
```

引用常用于函数参数，以避免拷贝并允许修改实参：

```cpp
void increment(int& x)
{
    ++x;
}
```

引用与指针的主要区别：
- 引用必须在定义时初始化，指针可以稍后赋值
- 引用不能为"空"，指针可以为 `nullptr`
- 引用一旦绑定就不能更改，指针可以重新指向其他对象
- 访问引用不需要解引用语法，访问指针需要 `*`

## 1.7.4 遍历

可以使用范围 `for` 语句遍历数组：

```cpp
int v[] = { 0, 1, 2, 3, 4, 5 };

for (auto x : v)            // 拷贝每个元素到 x
    std::cout << x << '\n';

for (auto& x : v)           // x 引用每个元素，可以修改
    ++x;
```

在范围 `for` 语句中，`auto` 让编译器自动推导元素类型。使用 `auto&` 可以避免拷贝并允许修改元素。

范围 `for` 语句不仅适用于数组，还适用于所有提供了 `begin()` 和 `end()` 的容器（如 `std::vector`、`std::list` 等），这将在后续章节中介绍。
