## 2.1 引言

C++ 的核心目标之一就是让程序员能够定义自己的类型，这些类型可以像内置类型一样方便、直观地使用。这种类型被称为*用户定义类型*（user-defined types，简称 UDT）。它们也被称为*类*（classes）和*枚举*（enumerations）。标准库中的大部分内容都是由用户定义类型构成的，而非语言内置特性。因此，理解用户定义类型是掌握 C++ 的关键。

通过定义新类型，程序员可以：

- 将相关的数据组织在一起（结构体）
- 控制数据的访问权限和不变式（类）
- 为一组相关的常量赋予有意义的名称（枚举）
- 在同一内存区域中存储不同类型的数据（联合体）

本章将逐一介绍这些机制。
