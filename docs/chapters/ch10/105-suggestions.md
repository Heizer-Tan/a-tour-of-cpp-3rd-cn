## 10.5 建议

1. 使用 `std::string` 而非 C 风格字符串来管理文本
2. 使用 `std::string_view` 作为只读字符串参数，避免不必要的拷贝
3. 注意 `string_view` 不拥有数据——确保底层数据在使用期间有效
4. 对于简单的模式匹配，优先使用 `string::find` 而非正则表达式
5. 使用原始字符串字面量 `R"(...)"` 编写正则表达式，避免转义混乱
6. 正则表达式编译代价较高——如果重复使用，将 `regex` 对象保存为 `static`
7. 使用 `std::to_string` 和 `std::stoi`/`std::stod` 进行数值与字符串的转换
8. 使用 `std::format`（C++20）进行类型安全的字符串格式化
