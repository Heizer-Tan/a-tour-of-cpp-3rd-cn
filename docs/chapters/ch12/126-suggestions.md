## 12.6 建议

1. 使用 `std::vector` 作为默认容器
2. 当需要频繁在中间插入/删除时，考虑 `std::list`
3. 当需要按键快速查找时，使用 `std::map` 或 `std::unordered_map`
4. 当需要有序遍历时使用有序容器，否则使用无序容器
5. 使用 `std::array` 替代 C 风格数组
6. 使用 `std::stack`、`std::queue` 表达特定的访问模式
7. 使用 `contains()`（C++20）检查元素是否存在
8. 使用结构化绑定（C++17）遍历 map
9. 为自定义类型提供哈希函数以用于无序容器
10. 了解每种容器的性能特征，根据需求选择
