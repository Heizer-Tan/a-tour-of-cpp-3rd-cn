## 19.2 历史

### 19.2.1 时间线

| 年份 | 里程碑 |
|------|--------|
| 1979 | "C with Classes" 开始开发 |
| 1985 | C++ 首次商业发布 |
| 1998 | C++98 标准（第一个 ISO 标准） |
| 2003 | C++03（小幅修订） |
| 2011 | C++11（重大更新：移动语义、lambda、auto、并发等） |
| 2014 | C++14（小幅改进：泛型 lambda、`make_unique` 等） |
| 2017 | C++17（结构化绑定、`optional`、`variant`、`filesystem` 等） |
| 2020 | C++20（概念、范围、模块、协程、`format` 等） |
| 2023 | C++23（`expected`、`std::print`、更多范围适配器等） |

### 19.2.2 设计哲学

C++ 的设计遵循以下原则：

- **零开销抽象**（zero-overhead abstraction）：你不用的东西，你不必为之付出代价
- **直接映射到硬件**：C++ 的数据模型直接对应机器架构
- **静态类型安全**：在编译时捕获尽可能多的错误
- **资源管理即初始化**（RAII）：将资源生命周期绑定到对象生命周期
- **泛型编程**：通过模板实现编译时多态
