# 索引

## 符号索引

- `#include`: 包含头文件
- `#define`: 宏定义
- `::`: 作用域解析运算符
- `.`: 成员访问运算符
- `->`: 指针成员访问运算符

- `catch`: 异常捕获
- `char`: 字符类型

- `const`: 常量限定符

- `default`: switch 默认分支
- `delete`: 删除对象
- `do`: do-while 循环
- `double`: 双精度浮点类型
- `else`: if 语句分支
- `enum`: 枚举类型
- `explicit`: 显式转换
- `export`: 导出模块
- `extern`: 外部链接
- `false`: 布尔假值
- `float`: 单精度浮点类型
- `for`: for 循环
- `friend`: 友元
- `goto`: 跳转语句
- `if`: 条件语句
- `inline`: 内联函数
- `int`: 整数类型
- `long`: 长整数类型
- `mutable`: 可变成员
- `namespace`: 命名空间
- `new`: 创建对象
- `noexcept`:  noexcept 说明符
- `nullptr`: 空指针
- `operator`: 运算符重载
- `private`: 私有访问
- `protected`: 保护访问
- `public`: 公共访问
- `register`: 寄存器变量

- `return`: 返回语句
- `short`: 短整数类型
- `signed`: 有符号类型

- `static_assert`: 静态断言
- `static_cast`: 静态转换
- `struct`: 结构体
- `switch`: 开关语句
- `template`: 模板

- `true`: 布尔真值

- `typedef`: 类型别名
- `typeid`: 类型信息

- `union`: 联合体
- `unsigned`: 无符号类型
- `using`: 使用声明/类型别名
- `virtual`: 虚函数
- `void`: 无类型
- `volatile`: 易变限定符
- `while`: while 循环

## 标准库索引

### 容器

- `vector`: 动态数组
- `list`: 双向链表
- `forward_list`: 单向链表
- `deque`: 双端队列
- `array`: 固定大小数组
- `map`: 有序关联容器
- `set`: 有序集合
- `unordered_map`: 无序关联容器
- `unordered_set`: 无序集合

### 算法

- `sort`: 排序
- `find`: 查找
- `transform`: 变换
- `accumulate`: 累加
- `for_each`: 遍历

### 字符串

- `string`: 字符串
- `string_view`: 字符串视图

### 智能指针

- `unique_ptr`: 独占指针
- `shared_ptr`: 共享指针
- `weak_ptr`: 弱指针

### 并发

- `thread`: 线程
- `mutex`: 互斥锁
- `condition_variable`: 条件变量
- `future`: 未来
- `promise`: 承诺
